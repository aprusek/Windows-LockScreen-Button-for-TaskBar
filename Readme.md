<h1>How to create a Windows Lock Screen Button</h1>

## Create the button yourself
- Create a Shortcut
- Put this text into the first dialog  "Rundll32.exe User32.dll,LockWorkStation"
- Name the shortcut in the second dialog eg "Lock Screen"
- Right Click on the shoortciut and select Properties
- Press the "Change Icon" and choose an icon
- Drag the shortcut into the Taskbar
